example_output_hg38_SQANTI3_Hub

This hub displays SQANTI3 transcriptome analysis results for the hg38 genome assembly.

üîç Advanced Filtering with bigBed 12+8

This hub uses bigBed 12+8 with an autoSql schema, enabling per-field filters and ranges:

  • Structural Category: FSM, ISM, NIC, NNC, genic, antisense, fusion, intergenic, genic_intron
  • Subcategory: mono-exon, multi-exon, novel, known, canonical, non-canonical
  • Coding Status: coding, non_coding, partial_coding, pseudo
  • FSM Class: A, B, C, D
  • Length: numeric range
  • Coverage: numeric range
  • Expression: numeric range

Use the track filters panel to filter by these fields. Text search also works on transcript IDs; Trix (if present) enables richer keyword search.

üé® Color Legend

  • Full-splice Match (FSM): Transcripts that perfectly match known genes
  • Incomplete-splice Match (ISM): Transcripts that partially match known genes
  • Novel In Catalog (NIC): Novel transcripts in known gene loci
  • Novel Not In Catalog (NNC): Novel transcripts in novel loci
  • Genic: Transcripts in gene regions
  • Antisense: Antisense transcripts
  • Fusion: Fusion transcripts
  • Intergenic: Intergenic transcripts
  • Genic Intron: Intronic transcripts

üöÄ Usage Instructions

To use this hub in the UCSC Genome Browser:

  1. Upload all files in this directory to a web-accessible location
  2. In the UCSC Genome Browser, go to My Data ‚Üí Track Hubs
  3. Enter the URL to your hub.txt file
  4. Select the appropriate genome assembly
  5. The SQANTI3 tracks will appear in your track list
  6. Right-click on the track and select "Filter" to access individual field filters
