üîç SQANTI3 Filtering Instructions

This track supports essential filtering for SQANTI3 transcriptome analysis in the UCSC Genome Browser!

How to Use Filters:

  1. Right-click on the track name in the UCSC Genome Browser
  2. Select "Configure" to access basic filter options (chrom, strand, score)
  3. Use "Filter by name" to search through enhanced transcript names containing all classification data
  4. Use "Filter by value" to search for specific classification values

Available Filter Categories:

Comprehensive SQANTI3 filtering through enhanced transcript names!

  • Standard BED Fields: chrom, strand, score (direct filtering)
  • Enhanced Names: All 48 SQANTI3 classification columns embedded in transcript names
  • Core Classification: structural_category, associated_gene, subcategory, coding, FSM_class, RTS_stage
  • Structure & Length: length, exons, intron_count, diff_to_TSS, diff_to_TTS
  • Coverage & Quality: min_cov, min_cov_pos, sd_cov, FL, all_canonical
  • Expression Data: iso_exp, gene_exp, ratio_exp, bite
  • Sequence Features: ORF_length, CDS_length, polyA_motif, predicted_NMD
  • Plus 20+ additional columns: Including all SQANTI3 quality metrics and annotations

Filter Features:

  • Range-based filtering for numeric values (length, coverage, expression)
  • Category-based filtering for classification data
  • Multiple filter combinations for complex queries
  • Search indexing for fast transcript name lookups
