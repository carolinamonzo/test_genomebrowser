This hub displays SQANTI3 transcriptome analysis results for the hg38 genome assembly.

üîç Advanced Filtering with Smart Name Encoding

This hub uses standard bigBed format with comprehensive data encoding in the name field:

  • Structural Category: Filter by FSM, ISM, NIC, NNC, genic, antisense, fusion, intergenic, genic_intron
  • Subcategory: Filter by mono-exon, multi-exon, novel, known, canonical, non-canonical
  • Coding Status: Filter by coding, non_coding, partial_coding, pseudo
  • FSM Class: Filter by A, B, C, D
  • Length: Range-based filtering for transcript length (L value)
  • Coverage: Range-based filtering for minimum coverage (C value)
  • Expression: Range-based filtering for isoform expression (X value)

Use the search box to filter by classification data (e.g., "intergenic", "L>500", "coding")

üîç How the Smart Filtering Works

All classification data is encoded in the transcript name field:

PB.118859.1|intergenic|mono-exon|non_coding|FSM1|L320|E1|Cnan|X80.0265

  • Format: transcript_id|structural_category|subcategory|coding_status|FSM_class|Llength|Eexons|Ccoverage|Xexpression
  • Search Examples:
    • intergenic - Find all intergenic transcripts
    • L>500 - Find transcripts longer than 500bp
    • E>2 - Find multi-exon transcripts
    • coding - Find all coding transcripts
    • FSM1 - Find FSM class A transcripts

This approach provides comprehensive filtering without the complexity of bigBed 12+ format!

üé® Color Legend

  • Full-splice Match (FSM): Transcripts that perfectly match known genes
  • Incomplete-splice Match (ISM): Transcripts that partially match known genes
  • Novel In Catalog (NIC): Novel transcripts in known gene loci
  • Novel Not In Catalog (NNC): Novel transcripts in novel loci
  • Genic: Transcripts in gene regions
  • Antisense: Antisense transcripts
  • Fusion: Fusion transcripts
  • Intergenic: Intergenic transcripts
  • Genic Intron: Intronic transcripts

üöÄ Usage Instructions

To use this hub in the UCSC Genome Browser:

  1. Upload all files in this directory to a web-accessible location
  2. In the UCSC Genome Browser, go to My Data ‚Üí Track Hubs
  3. Enter the URL to your hub.txt file
  4. Select the appropriate genome assembly
  5. The SQANTI3 tracks will appear in your track list
  6. Right-click on the track and select "Filter" to access individual field filters
